# 🧠 Sentient Website

A modern, responsive AI-themed website built with HTML, CSS, and JavaScript.

## 🚀 Features
- Clean and futuristic UI design
- Interactive chatbot
- Smooth animations
- Optimized for mobile and desktop
- Built with vanilla JS — no frameworks

## 🛠️ Tech Stack
- **HTML5**  
- **CSS3**  
- **JavaScript (ES6)**
